## :person_with_blond_hair: Как создать хорошего парня:
```javascript
var guy = document.body.appendChild (
    document.createElement ( 'good-boy' )
)
```
## :japanese_ogre: Как создать плохого парня:
```javascript
var scamp = document.body.appendChild (
    document.createElement ( 'good-boy' )
)
```
